Codex App Guide
หน้าแรก แผนการเรียนรู้ เริ่มต้น ความสามารถ เมนูอธิบาย Cheat Sheet อ้างอิง

Learning Path

แผนการเรียนรู้แบบเป็นขั้น

ออกแบบเพื่อให้เข้าใจ Codex App อย่างเป็นระบบ ลดความสับสน และใช้ได้จริงเร็วที่สุด

เหมาะกับใคร

มือใหม่ที่อยากเรียนรู้เร็ว, ทีมที่อยาก onboarding สมาชิกใหม่, หรือคนที่ไม่อยากอ่านทุกหน้าในครั้งเดียว

ภาพรวมเส้นทางการเรียนรู้

แบ่งเป็น 4 ช่วง เพื่อค่อยๆ เพิ่มความลึกแบบไม่ล้น

Phase 1

วันแรก: เริ่มต้นและโหมดพื้นฐาน

อ่านหน้าเริ่มต้น + ลองส่งคำสั่งแรกให้คล่อง

ไปที่หน้าเริ่มต้น
Phase 2

สัปดาห์แรก: เข้าใจฟังก์ชันหลัก

อ่านหน้าความสามารถ + เรียนศัพท์สำคัญในเมนูอธิบายฟังก์ชัน

ไปที่หน้าความสามารถ
Phase 3

สัปดาห์ที่สอง: ใช้จริงกับงานทีม

ฝึกใช้ Worktree, Diff/Review และการสรุปงานให้ทีม

ไปที่เมนูอธิบาย
Phase 4

สัปดาห์ที่สาม: ทำงานได้เร็วขึ้น

ใช้ Cheat Sheet และเริ่มตั้ง Automations หรือ Skills

ไปที่ Cheat Sheet

แผน 1-2-3 แบบย่อ

สำหรับคนอยากเริ่มเร็วที่สุด

1

เลือกงานสำคัญที่สุด

เริ่มจากงานที่ต้องทำจริง เช่น แก้บั๊ก หรือสรุปเอกสาร

2

เลือกโหมดและให้บริบท

Local / Worktree / Cloud และแนบไฟล์ที่เกี่ยวข้อง

3

ตรวจผลลัพธ์และสรุป

อ่าน diff และขอให้สรุปผล เพื่อส่งต่อทีม

ตารางฝึกใช้งาน 7 วัน
