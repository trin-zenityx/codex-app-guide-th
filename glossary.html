Glossary

เมนูอธิบายฟังก์ชัน

อ่านแบบแยกหมวด เปิด/ปิดได้เพื่อไม่ให้ตาลาย เหมาะกับมือใหม่ที่อยากเข้าใจศัพท์สำคัญ

วิธีใช้หน้านี้

กดเปิดเฉพาะหมวดที่ต้องใช้ แล้วค่อยกลับมาอ่านหมวดอื่นเมื่อพร้อม

โครงสร้างการทำงาน

Thread

ห้องสนทนาหนึ่งห้องสำหรับงานหนึ่งงาน เก็บบริบทและผลลัพธ์ของงานนั้นๆ

Agent

ผู้ช่วยอัตโนมัติที่ทำงานตามคำสั่ง เช่น เขียนโค้ด รีแฟกเตอร์ หรือสรุปรายงาน

Workspace / Project

โฟลเดอร์หลักของโปรเจกต์ที่ Codex เข้าถึงได้

Context / Auto Context

ข้อมูลประกอบที่ให้ Codex เข้าใจงาน เช่น ไฟล์ โฟลเดอร์ หรือข้อความ

โหมดการทำงาน

Local

รันงานในโฟลเดอร์ปัจจุบัน เหมาะกับงานแก้ไขเล็กๆ

Worktree

โฟลเดอร์ทำงานแยกจากงานหลัก ช่วยให้ทดลองไอเดียใหม่ได้ปลอดภัย

Cloud

รันงานบนสภาพแวดล้อมที่ตั้งค่าไว้ เหมาะกับงานใหญ่

Sandbox / Permissions

ขอบเขตการเข้าถึงไฟล์และคำสั่งเพื่อความปลอดภัย

การทำงานกับโค้ดและ Git

Diff / Review

ดูการเปลี่ยนแปลงโค้ดและคอมเมนต์ได้แบบ inline

Stage / Unstage

เลือกเฉพาะส่วนที่ต้องการส่งขึ้น Git

Commit / PR

สร้าง commit และเตรียม Pull Request ได้ในแอป

Worktree Branch

สร้างสาขาแยกเพื่อทำงานแบบไม่ชนกับงานหลัก

การขยายความสามารถ

Skills

ชุดคำสั่งหรือแนวทางเฉพาะที่เรียกใช้ซ้ำได้

Automations

งานอัตโนมัติที่ตั้งเวลาให้ทำซ้ำ เช่น รายงานรายวัน

Model Selection

เลือกโมเดลให้เหมาะกับงาน เช่น งานสรุปเร็วหรืองานละเอียด

System / User Instructions

กติกาการตอบและสไตล์งาน เช่น ภาษาและรูปแบบผลลัพธ์

อินพุตและเครื่องมือเสริม

Terminal ในเธรด

รันคำสั่ง ติดตั้งแพ็กเกจ หรือทดสอบได้ในเธรดเดียวกัน

Web Search

ค้นข้อมูลจากเว็บเพื่ออ้างอิงเอกสารล่าสุด

Image Input

แนบรูปเพื่อให้ Codex เข้าใจบริบท เช่น UI หรือกราฟ

Voice Dictation

สั่งงานด้วยเสียง แล้วแก้ไขข้อความก่อนส่งได้

การทำงานร่วมทีมและการติดตามผล

IDE Extension Sync

ซิงก์กับ IDE เพื่อให้บริบทตรงกัน

Notifications

แจ้งเตือนเมื่อเสร็จงาน

Prevent Sleep

กันเครื่องหลับระหว่างงานยาว

Activity Log

ย้อนดูสิ่งที่ Codex ทำไปแล้ว

อ้างอิงเอกสารทางการ

อ่านรายละเอียดเพิ่มเติมจากแหล่งข้อมูลทางการ
