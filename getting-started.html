หน้าแรก แผนการเรียนรู้ เริ่มต้น ความสามารถ เมนูอธิบาย Cheat Sheet อ้างอิง

เริ่มต้นแบบเร็ว

เริ่มต้นใช้งาน Codex App

เหมาะสำหรับมือใหม่ที่ต้องการเริ่มต้นให้ถูกทาง ตั้งแต่ติดตั้งจนส่งคำสั่งแรก

เป้าหมายของหน้านี้

ทำให้คุณเริ่มใช้งานได้ภายใน 10 นาที พร้อมเข้าใจโหมดหลักที่ต้องรู้

ขั้นตอน 1-2-3-4 แบบสั้นๆ

อ่านให้จบครั้งเดียวแล้วเริ่มได้ทันที

1

ติดตั้งและเข้าสู่ระบบ

ติดตั้งบน macOS แล้วเข้าสู่ระบบด้วยบัญชี ChatGPT หรือ API key

2

เลือกโปรเจกต์

เลือกโฟลเดอร์โปรเจกต์ที่ Codex จะเข้าถึงไฟล์ได้

3

เลือกโหมดการทำงาน

Local สำหรับงานเล็ก, Worktree สำหรับทดลอง, Cloud สำหรับงานหนัก

4

ส่งคำสั่งแรก

บอกเป้าหมาย + บริบท + รูปแบบผลลัพธ์ให้ชัด

Checklist ก่อนเริ่ม

เช็กให้ครบจะลดปัญหาเวลาทำงานจริง

ต้องมี

  • เข้าถึงโฟลเดอร์โปรเจกต์ได้
  • รู้ว่าอยากให้ Codex ทำอะไร
  • มีไฟล์/ข้อมูลประกอบที่เกี่ยวข้อง

ควรเตรียม

  • ข้อจำกัด เช่น ห้ามเปลี่ยน API
  • รูปแบบผลลัพธ์ที่ต้องการ
  • เวลาและงบประมาณงาน (ถ้ามี)

สูตรสั่งงานแบบง่ายที่สุด

ใช้ประโยคเดียวแต่ครอบคลุม: เป้าหมาย + บริบท + ข้อจำกัด + ผลลัพธ์

ตัวอย่าง

เป้าหมาย: แก้บั๊ก login
บริบท: ไฟล์ src/auth.ts
ข้อจำกัด: ห้ามเปลี่ยน API
ผลลัพธ์: สรุปเป็น bullet list พร้อมไฟล์ที่เปลี่ยน

ข้อผิดพลาดที่มือใหม่พบบ่อย

เลี่ยงสิ่งเหล่านี้จะทำให้ผลลัพธ์ตรงใจมากขึ้น

สั่งงานกว้างเกินไปโดยไม่มีบริบท
ไม่ระบุไฟล์/โฟลเดอร์ที่เกี่ยวข้อง
ไม่กำหนดข้อจำกัด เช่น ห้ามเปลี่ยน API
ไม่ตรวจ diff ก่อน commit
