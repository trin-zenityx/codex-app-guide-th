Codex App Guide
หน้าแรก แผนการเรียนรู้ เริ่มต้น ความสามารถ เมนูอธิบาย Cheat Sheet อ้างอิง

Capabilities

ความสามารถของ Codex App

ดูเป็นหมวดหมู่ พร้อมตัวอย่างใช้งานจริง เพื่อเลือกใช้ได้เร็ว

วิธีอ่านหน้านี้

ไล่ตามหมวดที่เกี่ยวข้องกับงานของคุณ ไม่จำเป็นต้องอ่านทั้งหมดในครั้งเดียว

หมวด: การทำงานแบบหลายงาน (Multi-thread)

เหมาะกับทีมที่ต้องทำหลายงานพร้อมกัน หรือคนที่อยากแยกงานให้ชัด

หลายเธรดหลายงาน

เปิดหลายเธรดได้พร้อมกัน และแต่ละเธรดมีบริบทของตัวเอง

ตัวอย่าง

งาน A: รีแฟกเตอร์ auth
งาน B: สรุปเอกสาร API
รันพร้อมกันได้คนละเธรด

Worktree

แยกโฟลเดอร์ทำงานเพื่อทดลองไอเดียใหม่ ลดความเสี่ยงกับโค้ดหลัก

ตัวอย่าง

สร้าง worktree สำหรับลอง UI ใหม่
ถ้าไม่เวิร์กก็ลบทิ้ง

หมวด: โหมดการทำงาน

เลือกโหมดให้เหมาะกับประเภทงานและความเสี่ยง

Local

งานเล็ก แก้เฉพาะจุด ใช้โฟลเดอร์หลักทันที

Worktree

ทดลองไอเดียใหม่ หรือแก้ระบบใหญ่โดยไม่ชนกับงานหลัก

Cloud

งานใหญ่หรือใช้ทรัพยากรสูง รันบนสภาพแวดล้อมที่จัดไว้

หมวด: Git และการส่งงาน

ฟังก์ชันที่ช่วยให้การตรวจงานและส่งงานปลอดภัยขึ้น

Diff + Review

ดูการเปลี่ยนแปลง พร้อมคอมเมนต์แบบ inline

Stage / Unstage

เลือกเฉพาะส่วนที่ต้องการส่ง

Commit / PR

เตรียม commit และสรุปสำหรับ PR ได้ในตัว

หมวด: เครื่องมือเสริมและอินพุต

เพิ่มบริบทให้ Codex เข้าใจงานเร็วขึ้น

Terminal ในเธรด

รันคำสั่ง ทดสอบ และตรวจผลได้ทันที

Web Search

ค้นข้อมูลจากเว็บและสรุปให้ทีม

Image Input

แนบภาพ UI หรือกราฟเพื่อให้เข้าใจบริบท

Voice Dictation

พูดเพื่อพิมพ์คำสั่ง แล้วค่อยแก้ไขก่อนส่ง

ต่อไปอ่านอะไรดี?

ถ้าอยากเข้าใจศัพท์ทั้งหมด ให้ไปหน้าเมนูอธิบายฟังก์ชัน

ไปที่เมนูอธิบาย เปิด Cheat Sheet

ZenityX (สถาบันเซนนิตี้เอ็กซ์)
